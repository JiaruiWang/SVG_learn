# SVG_learn
##SVG的打开方式
 - 浏览器直接打开
 - `<img>`标签引用
 - 在`HTML`中使用`svg`标签
 - 作为`CSS`背景

##基本图形和属性
- rect, circle, ellipse line polyline polygon
- fill stroke stroke-width transform

##基本API
 - document.createElementNS(ns, tagName)
 - element.appendChild(childElement)
 - element.setAttribute(name, value)
 - element.getAttribute(name)
